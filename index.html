Helm chart for the Cloudflare Web Application Firewall log downloader

Install the Helm chart in your cluster

helm repo add waf https://mihaibojin.github.io/waf-downloader

helm install -f config.yaml waf-downloader waf/waf-downloader

You must also provide several configuration values, e.g.:

                    # config.yaml
                    cloudflare:
                      apiToken:  # Specifies the Cloudflare API token
                      zoneId:  # Specifies the zone to download logs from
                    outputs:
                      dbConnStr:  # Specifies a Postgres endpoint to send the logs to
